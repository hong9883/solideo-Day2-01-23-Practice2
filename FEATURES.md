### 추천 알고리즘

#### 점수 계산 시스템
```typescript
// 여행지 추천
score = 0
- 여행 스타일 일치: +10점/개
- 예산 범위 일치: +5점
- 기본 평점: +2점 × 평점

// 레스토랑 추천
score = 0
- 음식 취향 일치: +15점/개
- 예산 범위 일치: +8점
- 기본 평점: +3점 × 평점
```

#### 필터링 기준
1. **예산 수준**
   - 저예산: 1-2레벨 장소
   - 중간: 2-3레벨 장소
   - 고예산: 3-4레벨 장소
   - 럭셔리: 4레벨 장소

2. **거리 기반**
   - 도착지 중심 반경 내 추천
   - 가까운 순으로 정렬 옵션 (향후)

3. **평점 기반**
   - 최소 평점 4.0 이상
   - 리뷰 수 고려 (향후)
